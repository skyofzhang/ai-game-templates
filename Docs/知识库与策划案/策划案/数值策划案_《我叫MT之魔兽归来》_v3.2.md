## 1. 数值设计原则

> **原则**: 这是所有数值设计的最高"宪法"，后续所有数值配置都必须严格遵守。

### 1.1 核心设计原则

**成长曲线原则**: 玩家的成长应该是平滑的，避免断崖式的难度跳跃

**挑战与奖励原则**: 挑战越大，奖励越丰厚

**数值护栏原则**: 所有数值都应该在合理的范围内，避免数值膨胀

**参考标准**: 《迷雾大陆》的数值设计（平滑成长、合理挑战、丰厚奖励）

### 1.2 数值护栏

**玩家属性护栏**:
- **HP**: 100 ~ 10,000
- **Attack**: 10 ~ 1,000
- **Defense**: 5 ~ 500
- **MoveSpeed**: 3 ~ 10 (米/秒)

**怪物属性护栏**:
- **HP**: 50 ~ 50,000
- **Attack**: 5 ~ 2,000
- **Defense**: 0 ~ 1,000
- **MoveSpeed**: 1 ~ 8 (米/秒)

**装备属性护栏**:
- **HP加成**: +10 ~ +500
- **Attack加成**: +5 ~ +200
- **Defense加成**: +3 ~ +100

### 1.3 数值计算公式

**伤害计算公式**:
```
最终伤害 = max(1, 攻击者Attack - 防御者Defense) × 技能倍率 × 暴击倍率
```

**暴击计算公式**:
```
暴击率 = 基础暴击率(5%) + 装备暴击率加成
暴击倍率 = 1.5 ~ 2.0
```

**经验计算公式**:
```
升级所需经验 = 100 × (当前等级)^1.5
```

**金币掉落计算公式**:
```
金币掉落 = 怪物基础金币 × (1 + 随机浮动[-20%, +20%])
```

### 9.3 经济平衡

**平衡参数**:
- 完成第1关后，玩家应该有约150金币，可以购买1瓶治疗瓶
- 完成第5关后，玩家应该有约3,000金币，可以购买30瓶治疗瓶或1件稀有装备
- 完成第10关后，玩家应该有约15,000金币，可以购买150瓶治疗瓶或3件史诗装备
