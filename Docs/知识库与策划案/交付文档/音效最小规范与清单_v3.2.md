# 音效最小规范与清单 v3.2（首版自包含）

> **目的**：替代“待建立音效知识库后补充”的外部依赖疑问，使交付包在**首版**范围内对音效也可直接落地。  
> **范围**：只覆盖首版最小可用（MVP）音效，不扩展系统。  
> **日期**：2026-01-30  

**与《验收与口径》4.3 音频 Key 的对应**：本文档为首版音效 key 的**唯一来源**；《验收与口径》4.3 规定「任意播放音效的 key 须来自该清单」。程序/UI 中使用的 BGM/SFX key 须与本清单一致，缺资源时允许占位/复用，但必须不崩溃。

---

## 1) 资源规范（落地约束）

- **文件格式**：`.wav`（推荐）或 `.ogg`
- **采样率/位深**：44.1kHz / 16-bit（默认即可）
- **声道**：SFX 单声道优先；BGM 立体声
- **响度建议**：避免过度压限；同类音效保持相对一致（例如按钮音效不要比普攻命中更响）
- **命名规则**：
  - BGM：`BGM_<SceneOrContext>.wav`（例：`BGM_MainMenu.wav`）
  - SFX：`SFX_<ActionOrContext>.wav`（例：`SFX_ButtonClick.wav`）
- **路径建议（Unity）**：
  - `Assets/Audio/BGM/`
  - `Assets/Audio/SFX/`

> **实现建议（非强制）**：`AudioManager` 以 **string key** 播放（例如 `PlayBGM("BGM_MainMenu")`、`PlaySFX("SFX_ButtonClick")`），key 与文件名去扩展名保持一致。

---

## 2) 首版最小清单（必须提供）

### 2.1 BGM（2条）

| Key | 用途 | 触发时机 | 建议时长 |
|---|---|---|---|
| BGM_MainMenu | 主界面背景音乐 | 进入主界面 | 60-120s 循环 |
| BGM_Gameplay | 战斗背景音乐 | 进入战斗关卡 | 60-120s 循环 |

### 2.2 SFX（8条）

| Key | 用途 | 触发时机 | 备注 |
|---|---|---|---|
| SFX_ButtonClick | 按钮点击 | 任意可点击按钮按下 | UI 通用 |
| SFX_Attack | 普攻命中 | 玩家普攻命中怪物 | 可短促 |
| SFX_Hit | 受击 | 玩家或怪物受击 | 可复用 |
| SFX_Death_Monster | 怪物死亡 | 怪物死亡 | 可偏“碎裂/消散” |
| SFX_Death_Player | 玩家死亡 | 玩家死亡 | 可更明显 |
| SFX_Skill | 技能释放 | `SK001/SK002` 释放时 | 首版可共用 1 条 |
| SFX_LevelUp | 升级 | 角色升级 | UI/反馈 |
| SFX_GoldPickup | 拾取金币 | 拾取金币 | 反馈清脆 |
| SFX_Pickup | 拾取物品（通用） | 拾取道具/装备等 | 可与 SFX_GoldPickup 二选一或复用 |

---

## 3) 占位规则（允许缺资源但必须不崩溃）

- **允许占位**：首版资源未齐时，可用静音或同类音效复用占位
- **禁止崩溃**：播放音效找不到资源时必须降级处理（日志可警告，但不应中断流程）
